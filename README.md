# 🎬 Movie Ticket Booking System

A modern, interactive **cinema ticket booking web application** built using **HTML, CSS, and JavaScript**.
This project simulates a real-world movie booking experience with realistic seat selection, tier-based pricing, and smooth UI interactions.

---

## 🚀 Features

### 🎥 Movie Selection

* Displays **5 movie posters**
* Selected movie moves to focus
* Other movies are slightly dimmed for better UX

### 💺 Realistic Seat Layout

* Cinema-style seating arrangement
* Includes **aisle gaps**
* Large seating layout for better realism

### 🏷️ Tier-Based Pricing

Seats are divided into tiers:

* 🥇 Gold — ₹300
* 🥈 Silver — ₹220
* 🥉 Regular — ₹150

Pricing is automatically calculated based on selected seats.

### 🔢 Seat Selection System

* Users select number of seats using **+ / − buttons**
* Cannot exceed selected seat limit
* Interactive seat selection with visual feedback

### 🎨 Seat Status Indicators

* 🟫 Available → Gray
* 🔵 Selected → Blue
* 🔴 Booked → Red

### 🎥 Cinematic UI

* Dark theme design
* **Glowing cinema screen** at the bottom
* Smooth transitions and animations

### 🪟 Checkout System

* Modal popup showing booking summary
* Includes:

  * Movie name
  * Selected seats
  * Total price

### 🎉 Booking Confirmation

* Full-screen confirmation message
* Auto disappears after 2 seconds
* Booked seats remain marked in red

### ✨ Toast Notifications

* Animated messages for:

  * Seat selection
  * Errors (seat limit, no selection, etc.)
